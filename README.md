# 💳 Gestionnaire de Chèques Pro v3.0

## 🎯 Description

Système complet de gestion de chèques avec interface moderne développé en Python/Tkinter. 
Application professionnelle pour le suivi, la gestion et l'analyse des chèques avec fonctionnalités avancées.

## ✨ Fonctionnalités Principales

### 🏦 Gestion Complète
- **Banques et Agences** : Gestion complète des établissements bancaires
- **Clients** : Support des personnes physiques et entreprises
- **Chèques** : Suivi complet avec statuts et échéances
- **Utilisateurs** : Système de rôles (Admin, Comptable, Agent, Lecture seule)

### 📊 Tableau de Bord
- Statistiques en temps réel
- Graphiques interactifs (statuts, évolution temporelle)
- Alertes et notifications automatiques
- Indicateurs de performance

### 🔍 Recherche et Filtrage
- Recherche textuelle avancée
- Filtres multiples (statut, banque, période, montant)
- Tri et regroupement des données
- Export des résultats filtrés

### 📈 Rapports et Analyses
- Résumé général avec graphiques
- Analyse temporelle (mensuelle, hebdomadaire, quotidienne)
- Performance par banque
- Top clients et statistiques
- Export PDF/Excel/CSV

### 🔔 Notifications
- Rappels automatiques d'échéance
- Notifications de changement de statut
- Centre de notifications centralisé
- Alertes personnalisables

### 💾 Sauvegarde et Export
- Sauvegarde automatique et manuelle
- Export multi-formats (PDF, Excel, CSV)
- Historique des exports
- Restauration de données

## 🛠️ Technologies Utilisées

- **Python 3.8+**
- **Tkinter** - Interface utilisateur
- **SQLite** - Base de données
- **Matplotlib** - Graphiques et visualisations
- **ReportLab** - Génération PDF
- **OpenPyXL** - Export Excel
- **tkcalendar** - Sélecteur de dates
